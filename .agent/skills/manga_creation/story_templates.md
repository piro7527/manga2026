# ストーリーテンプレート集

4コマ漫画の起承転結パターン集。各テンプレートをベースにアレンジして使う。

---

## 1. テクノロジーあるある 🖥️

**参考**: デジタル教科書の話

| コマ | 内容                                     |
| ---- | ---------------------------------------- |
| 起   | グレにゃんが従来の方法で苦労している     |
| 承   | さらに苦労が深まる描写                   |
| 転   | シロにゃんが登場、テクノロジーで解決済み |
| 結   | グレにゃんの反応（羨望・呆然・皮肉）     |

**ストーリー例**: 紙の地図 vs GPS、手書きノート vs タブレット、現金 vs 電子マネー

---

## 2. 季節イベント系 🌸

| コマ | 内容                               |
| ---- | ---------------------------------- |
| 起   | 季節のイベントに張り切るグレにゃん |
| 承   | 準備・期待を膨らませる             |
| 転   | 予想外のハプニング発生             |
| 結   | オチ（切ない・笑える結末）         |

**ストーリー例**: 花見で場所取り→雨、海水浴→日焼け、クリスマス→ぼっち

---

## 3. 朝の通学・通勤 🏫

| コマ | 内容                                       |
| ---- | ------------------------------------------ |
| 起   | 眠そうに登校するグレにゃん                 |
| 承   | 遅刻しそうで焦る、または面倒な出来事       |
| 転   | 意外な展開（実は休校日、曜日を間違えた等） |
| 結   | 脱力オチ                                   |

**ストーリー例**: 休日に登校、忘れ物で往復、電車で寝過ごし

---

## 4. 食べ物ネタ 🍙

| コマ | 内容                             |
| ---- | -------------------------------- |
| 起   | 美味しそうな食べ物を前にワクワク |
| 承   | いざ食べようとする               |
| 転   | 邪魔が入る or 予想外の事態       |
| 結   | 食べられない or 別の展開         |

**ストーリー例**: 限定スイーツが売り切れ、弁当を忘れる、辛すぎるカレー

---

## 5. 勉強・試験 📝

| コマ | 内容                               |
| ---- | ---------------------------------- |
| 起   | 試験前夜、余裕の表情               |
| 承   | いざ勉強開始…のはずが脱線          |
| 転   | 気づいたら朝 or 全然違う教科を勉強 |
| 結   | 絶望のオチ                         |

**ストーリー例**: SNSに夢中で勉強せず、範囲を間違える、徹夜明けで試験中に寝る

---

## 6. 買い物 🛒

| コマ | 内容                           |
| ---- | ------------------------------ |
| 起   | 必要なものを買いに出かける     |
| 承   | 目的のものを探す or 店に到着   |
| 転   | 全然違うものを買ってしまう     |
| 結   | 家に帰って気づく or 次回もやる |

**ストーリー例**: コンビニで余計なもの、ネット通販の衝動買い、セールの罠

---

## 7. 天気・天候 ☔

| コマ | 内容                                 |
| ---- | ------------------------------------ |
| 起   | 天気予報を確認する                   |
| 承   | 予報を信じて行動する                 |
| 転   | 完全に予報が外れる                   |
| 結   | ずぶ濡れ or 逆にシロにゃんは準備万端 |

**ストーリー例**: 傘を持たずに出て土砂降り、晴れ予報で厚着して猛暑

---

## 8. SNS・スマホ 📱

| コマ | 内容                                |
| ---- | ----------------------------------- |
| 起   | SNSで見かけた情報にテンション上がる |
| 承   | 実際に試してみる                    |
| 転   | 現実はそう甘くない                  |
| 結   | SNSと現実のギャップに絶望           |

**ストーリー例**: 映えスポットが混雑、簡単レシピが全然簡単じゃない、バズり狙い失敗

---

## 9. 運動・健康 🏃

| コマ | 内容                       |
| ---- | -------------------------- |
| 起   | 「今日から運動する」と宣言 |
| 承   | 意気揚々と準備する         |
| 転   | 3分で挫折 or 思わぬ筋肉痛  |
| 結   | 「明日から本気出す」       |

**ストーリー例**: ジム入会→行かない、ランニング→すぐ歩く、ダイエット→ご褒美に食べる

---

## 10. 対比オチ（グレにゃん vs シロにゃん）🐱

| コマ | 内容                                   |
| ---- | -------------------------------------- |
| 起   | ある状況でグレにゃんの行動（非効率）   |
| 承   | グレにゃんがさらに苦労する             |
| 転   | シロにゃんの同じ状況での行動（効率的） |
| 結   | 2匹の対比ショット。無言 or 一言のオチ  |

**ストーリー例**: アナログ vs デジタル、早起き vs 夜型、計画的 vs 行き当たりばったり

---

## テンプレートの使い方

1. 上記からテーマに近いテンプレートを選択
2. 具体的なストーリーに置き換える
3. 各コマのセリフ・動作を具体化する
4. `SKILL.md` のフローに従って画像を生成する
