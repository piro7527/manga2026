# 長い休み
作成日: 2026-03-21

## コマ割り

### 1コマ目（起）
- 場面: ドライブ中。1960年代のFiat 500が走っている。車から音符マークが飛び出している。
- キャラ: グレにゃん（車から身を乗り出さずに、車内でニコニコ笑顔で運転中）
- セリフ/テキスト: 楽しげな手書き風記号（音符マーク）

### 2コマ目（承）
- 場面: 夜中の部屋。ソファーと机がある。机の上には食べたお菓子の袋とジュースのコップ。
- キャラ: グレにゃん（だらしなくソファーに座り、お菓子を食べながら笑っている）
- セリフ/テキスト: 映画の音声風の手書き記号、お菓子の袋のパッケージ文字（手書き記号）

### 3コマ目（転）
- 場面: ハンバーガーショップ。机の上に食べかけのハンバーガーとコーラ。
- キャラ: グレにゃん（コーラをガブガブ飲んでいる）、友達の猫たち（三毛猫、とら猫など。一緒に集まっている）
- セリフ/テキスト: 友達とわいわい話している風の手書き記号

### 4コマ目（結）
- 場面: 久しぶりの再会シーン。何もないシンプルな背景。
- キャラ: 大きく太ってしまったが、何事もなくニコニコ笑顔のグレにゃん。それを見て「やっぱりか・・・」と冷めた目を向けるシロにゃん。
- セリフ/テキスト: シロにゃんの心の声（もくもく吹き出しの中に手書き記号）
- タイトル: 「長い休み」を右下などの枠内に手書き風フォントで配置。
